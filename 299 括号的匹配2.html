字典的构建

python中的字典是哈希表的典型实现，可以使用字典来实现括号的匹配

map={
    ")":"(",
    "]":"[",
    "}":"{",
}

括号匹配的实现

使用list可以获得一个栈，在字符temp不在字典中时，直接使用append进行入栈，否则进行匹配

def isValid(s:str):
    map={
        ")":"(",
        "]":"[",
        "}":"{",
    }
    stack=list()
    for temp in s:
        if(temp in map):
            if(not stack or stack[-1]!=map[temp]):
                return False
            else:
                stack.pop()
        else:
            stack.append(temp)
    return not stack
